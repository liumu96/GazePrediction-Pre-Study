# EgoBody Notebooks

Interactive notebooks for EgoBody visualization and analysis.

- `13_egobody_missing_gaze_interactive.ipynb`: Interactive missing-frame gaze
  direction comparison for SparseGaze residual and HAGI++. It shows angular
  error plus CPF pitch/yaw timelines. It intentionally does not draw 3D rays or
  scanpath endpoints because the local EgoBody copy has no RGB/depth/scene-hit
  points.
