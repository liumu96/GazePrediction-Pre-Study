# ADT Notebooks

Interactive notebooks for ADT visualization and analysis.

- `02_gaze_head_scene_viewer.ipynb`: Matplotlib 3D gaze/head direction sanity check.
- `04_gaze_head_scene_viewer_interactive.ipynb`: Plotly 3D gaze/head direction viewer.
- `05_scene_object_gaze_viewer.ipynb`: Static scene object, skeleton, head, and gaze viewer.
- `06_scene_object_gaze_dynamic_viewer.ipynb`: Dynamic frame-by-frame scene viewer.
- `07_multiview_gaze_dashboard.ipynb`: Multi-view gaze, motion, image, object-hit, and 3D dashboard.
- `08_prediction_gaze_evaluation_viewer.ipynb`: SparseGaze prediction diagnostics.
- `09_npz_gaze_visualization_viewer.ipynb`: Per-sequence prediction NPZ visualization.
- `10_adt_hagi_sparsegaze_compare.ipynb`: ADT HAGI++ vs SparseGaze comparison.
- `11_scene_scanpath_3d_interactive.ipynb`: Interactive scene-space scanpath comparison.
- `12_scene_scanpath_3d_animation.ipynb`: Dynamic scene-space scanpath animation.
